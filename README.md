# Tetris (Pygame)

Simple Tetris implemented with Pygame.

Requirements
- Python 3.8+
- See `requirements.txt` (install with pip)

Run
```bash
pip install -r requirements.txt
python main.py
```

Controls
- Left / Right: move
- Down: soft drop
- Up: rotate
- Space: hard drop
- P: pause
- Esc or close window: quit
